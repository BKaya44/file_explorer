# Rust File Explorer

This project was created to help me learn more about Rust. It is a file explorer designed for Windows 11 as an alternative option.

## Requirements

-   [Rust](https://rust-lang.org/tools/install/)

## Building

```bash
cargo build
cargo run
```

## Features

- Explore folders
- Open files
- Add folder to bookmarks
- Remove folder from bookmarks

### Planned features

- More options to `right click` menu
- Remove bookmark when folder is deleted (currently only removed when app loads)
- Shortcuts
- Better UI design
- Icons
- Settings
- File/folder searching
